# Interactive Line Art Earth

A beautiful, interactive 3D line art visualization of Earth using Three.js.

## 🌐 Live Demo

Visit: [https://rinshannkaihou.github.io/mono_earth/](https://rinshannkaihou.github.io/mono_earth/)

## ✨ Features

- Interactive 3D globe with line art style
- Smooth mouse/touch controls for rotation
- Minimalist aesthetic with paper texture background
- Responsive design

## 🛠 Built With

- [Three.js](https://threejs.org/) - 3D graphics library
- [Tailwind CSS](https://tailwindcss.com/) - Styling
